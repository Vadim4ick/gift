# Почему MDX — лучший формат для блога в Next.js

Если у тебя уже есть проект на **Next.js** и ты хочешь добавить блог **без базы данных и сервера**,  
MDX — почти идеальный вариант.

Он позволяет писать статьи как обычный Markdown,  
но при этом **вставлять React-компоненты прямо в текст**.

---

# 234234234

## Что такое MDX

**MDX** — это Markdown + JSX.

Проще говоря:

- текст → как в Markdown
- интерактив → как в React
